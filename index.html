Connect Firebase
Nexus stores your encrypted messages in your own Firebase Realtime Database. You own the data — we never touch it.
  1. Go to console.firebase.google.com
  2. Create a new project (or open an existing one)
  3. Click Add app and choose Web (</>)
  4. Register the app — you can skip Firebase Hosting
  5. Copy the firebaseConfig object shown on screen
  6. In the left sidebar, go to Build → Realtime Database
  7. Click Create Database, choose a region, start in locked mode
  8. Go to Rules and paste:

    { "rules": { "accounts": { "$uid": { ".read": "$uid === auth.uid", ".write": "$uid === auth.uid", "messages": { ".indexOn": ["timestamp"] } } } } }

    Then click Publish
  9. In Authentication → Sign-in method, enable Google
  10. Paste your config below
